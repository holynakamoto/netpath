- [01KWF7SYJ5WQR4Z46PPDB1YD4K] **- **`concurrent** -- - **`concurrent.futures.ThreadPoolExecutor` chosen over asyncio for `_measure()` concurrency** — A single `ThreadPoolExecutor(max_workers=8)` wraps the entire `_measure()` body via context manager; all futures (v4/v6 dual-stack traces, PMTU, TCP connect, TLS handshake, RUM fetch, ping during iperf3) are submitted to this one executor. The context manager guarantees structured teardown — all futures are joined when `_measure()` exits, including early-return paths. Futures that time out are cancelled and populate `probe_errors`.
